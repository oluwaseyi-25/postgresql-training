## Bootcamp Overview

**Goal:** Take learners from zero to advanced PostgreSQL skills, including real-world use cases, performance optimization, and production-ready practices.

**Target Audience:** Engineers, Analysts, Data Professionals, Backend Developers.

**Structure:** Each week contains:
- Learning objectives
- Slides / lesson notes
- Hands-on labs
- Homework assignments
- Reference solutions
